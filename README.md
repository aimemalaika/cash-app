## Getting Started
- clone the repository by running\
    `git https://github.com/aimemalaika/blog.git`
- navigate to the folder\
    `cd blog`
- Install packages\
    `bundle install`
- Setup database\
    `rails db:setup`
- Run migration\
    `rails db:migrate`
